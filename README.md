Golden Piano Waterfall is a piano performance visualization app based on MIDI files, written in HTML and JavaScript. The app allows you to:
Load and play MIDI files with a stylish "waterfall" notes animation.
Adjust visual parameters, track colors, particle styles, size, transparency, and other effects.
Switch between day/night mode, demo/playback, customize visuals, save snapshots, and record to WebM.
Change tempo and control MIDI.
Advanced effects like shadows, rainbow mode, highlighted keys, ghost notes, and more.
Technologies: HTML, CSS, JavaScript (with the Tone.js library for sound and MIDI playback).
You can post a tutorial about this program on social media.
